## Опис

**МайндКер** — десктопний застосунок на Python (Tkinter) для онлайн-психологічної підтримки з вбудованим ШІ-агентом на основі правил.

---

## Реалізований функціонал (v1.1)

| Модуль | Опис |
|--------|------|
| 🔐 Реєстрація / Вхід | Валідація імені, перевірка формату пошти, унікальність |
| 🗨️ Чат-сесія | Розмова з ШІ-агентом, вибір теми, автозбереження |
| 📋 Історія сесій | Перегляд попередніх розмов із деталями |
| 📤 Експорт JSON | Структурований звіт із метаданими та всіма сесіями |
| 📊 Експорт CSV | Таблиця сесій для аналізу у Excel / LibreOffice |
| 📝 Журналювання | Всі дії та помилки → `logs/app.log` |
| ⚙️ Конфігурація | Зовнішній файл `config/settings.json` |
| ✅ Валідація | Зрозумілі повідомлення про помилки для новачків |
